topsDetailPanel Class Reference

Overview

Superclass to manage each reusable, movable uipanel portion of a topsGUI graphical interface.

topsDetailPanel relies on its parentGUI, an instance of topsGUI or a subclass, to function. It's not able to function independently like a regular uipanel.

Color in string variables using standard colors.

Parameters:
string a string that should be colored in

Computes the "sum" of the string and uses it as an index into the standard color map. Returns the color from the color map.

The idea is that strings should pop out visually, and identical strings should pop out together and be memorable from gui to gui.

Present standard controls for editing values.

Parameters:
getter fevalable cell array to invoke that returns the value to be displayed
setter fevalable cell array to invoke to set a new value from user input (should expect the new value as the first argument).

Returns a list of standard arguments to show and allow editing of a value. The value displayed will reflect the return value of the getter fevalable.

The user may click inside the control to type a new string, which will be passed to Matlab's built-in eval() function. The eval() result wil be passed to the setter function to update the value.

The control will not attempt to validate user inputs. It will attempt to catch errors and display them, and only invoke the setter upon success.
