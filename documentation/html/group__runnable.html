Property allParameters = struct([]) [inherited]

struct array of parameter names, values and assignment targets

Each field name of allParameters is the name of an arbitrary parameter. Each field contains another struct with the fields:

  • values cell array of possible values for the parameter
  • assignments struct array of objects and substruct()-style references used to assign parameter values to each object.

Users should call addParameter() and addAssignment() instead of manipulating allParameters directly.

which ensemble objects = {} [protected, inherited]

Get a property value for one or more objects.

array of objects in the ensemble

Parameters:
property string name of an ensemble object property
index optional ensemble object index or indexes

Gets the value of the named have in common. Returns a cell array with one element per object, containing each property value. If there is only one object, returns the value without enclosing it in a cell array.

By default, gets a value from each each ensemble object. If index is provided, it may specify a subset of ensemble objects.
