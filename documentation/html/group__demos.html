Set up the SquareTag demo task, with user inputs.

Parameters:
logic SquareTagLogic object, the "back end" of SquareTag
av SquareTagAV object, the audio-visual "front end" of SquareTag
getXFunction function that returns x-position, as from a mouse
getYFunction function that returns y-position, as from a mouse

Set up to play SquareTag interactively. Uses logic and av as the "back end" and "front end" of the task, respectively. If either is omitted, configureSquareTag makes defaults. getXFunction and getYFunction must be functions_handles. Each funciton should return an x- or y-position input by the subject, for example from a mouse or joystick. x- and y-values should be scaled to the unit square, so that left and bottom are at 0 and top and right are at 1.

Returns a topsRunnable object which can be run() to start playing SquareTag. Also returns as a second output a topsGroupedList which contains data and objects related to the SquarTag session, including logic, av, getXFunction, and getYFunction.

demoSquareTag ( logic  ,
av  ,
tagSteps   
)

Demonstrate the SquareTag task, without user input.

Parameters:
logic SquareTagLogic object, the "back end" of SquareTag
av SquareTagAV object, the audio-visual "front end" of SquareTag
tagSteps number of frames to tag each square programmatically

demoSquareTag() runs through the SquareTag demo task without user intervention. It coordinates the behaviors of the given logic and av objects to make a complete task-like demo. If logic or av is omitted, demoSquareTag() creates a default object. Instead of accepting inputs from a user or subject, demoSquareTag() moves the subject's cursor programmatically, over the given tagSteps number of animation frames.
