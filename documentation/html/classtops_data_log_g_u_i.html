topsDataLogGUI Class Reference
[Foundation]

Overview

Visualize data sorted by time.

topsDataLogGUI plots a summary of data event times from the current topsDataLog, as a raster over time. You can show and hide different data groups. You can view different time ranges.

There will be more to come, as I reimagine this GUI...
